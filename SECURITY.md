# EggyUI 安全策略

## 支持版本
EggyUI 的旧版本已不受支持，请使用新版 EggyUI。
|版本|支持|
|----|----|
| 1.0 RE | ✅ |
| 3.5 | ❌ |
| 3.0 | ❌ |
| 2.5 | ❌ |
| 2.0 | ❌ |
| 1.0 | ❌ |
