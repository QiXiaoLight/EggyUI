# EggyUI 行为准则

本准则适用于EggyUI项目的所有代码仓库、讨论区、Issue及Pull Request。参与本项目的开发即视为同意遵守以下约定。

## 一、代码与内容

1. **不拆包，不侵权**  
   本项目不使用任何游戏拆包/解包资源。所有视觉素材均来自合规渠道或团队重制。ExRFy拆包蛋仔派对事件已证明：侵权传播的尽头是平台处罚与法律追责。我们不欢迎任何与拆包相关的内容、链接或讨论。

2. **不捆绑，不留坑**  
   安装包不捆绑任何未经用户知晓的软件。卸载即清理，不留注册表残留、不写系统目录。如果你发现任何组件违反此原则，请提交Issue，我们会立即处理。

## 二、交流与协作

1. **就事论事，不站队**  
   Issue区只讨论技术问题，不讨论个人恩怨。苏陌团、欣彩自媒团等外部社群的内部矛盾，与本项目无关。不要在Issue、PR或讨论区中拉帮结派、挂人、翻旧账。

2. **不引流，不挂人**  
   不要发布指向其他社群矛盾的链接或截图。如果你在其他地方有纠纷，请留在那里解决。本项目不是仲裁庭，不是扩音器，不是避难所。

## 三、执行

违反本准则的内容将被直接关闭或删除，屡次违规者将被封禁访问权限。不警告，不辩论。
